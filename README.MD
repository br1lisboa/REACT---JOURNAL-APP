# JOURNAL APP - REACT

Application in which we configure Redux, applied at the beginning in the authentication and maintaining the state throughout the entire application.

## Installation

Use the package manager [yarn](https://yarnpkg.com/).

```bash
yarn install
```

```bash
yarn dev
```

## Characteristic
*Redux*

*Firebase*

*Fire Store*

*Redux Devtools*

*Thunk*

*Forms*

*Google SignIn*

*Persistent state*
